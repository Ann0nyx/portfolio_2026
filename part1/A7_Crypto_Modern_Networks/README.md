# A7 – Cryptography in Modern Networks

### Cryptography Identified in Modern Networks – WPA2 encryption over network traffic

### Explanation - 
One such Cryptography used in Modern Networks is Wifi encryptions such as WPA2 or WPA3. These protocols secure the wireless communication between and network routers by encrypting data transmitted over the network, making sure sensitive information like login credentials , personal data or logs cannot be intercepted by attackers.When a device connects to a wifi network, it first authenticates it using a password. Once authenticated,an encrypted channel is made between the device and the router. This encryption converts all transmitted data into unreadable format, preventing attackers from accessing or modifying the information.

#### What I learned –
When connecting to my home WIFI network, I need to enter a password before accessing it. Further when connected, all activities such as web browsing, accessing LMS or even online apps are encrypted through the WIFI protocol,securing communication. It can be identified by looking through the network settings in my laptop. Additionally I learned that network protocols like WPA2 or WPA3 provides stronger protection for everyday internet usage.
