# A6 – Cryptographic Implementation used Offline

I found that a regular Cryptographic implementation I use Offline are the password protected pdf’s they are relatively easy to encrypt and is widely used in my work place medical clinic to encrypt patient files. It secures documents stored locally on a device by requiring a password to access the file.

### How I implemented it – 

I used the preview app in my mac to open a pdf document that I wanted to encrypt. Then I exported it and opened file permissions which lets me apply a password to the document and other permissions as necessary. The file is then saved and when reopened it asks for a password which if entered incorrectly denies access to the document.

Through this activity I learned that cryptography can be applied to protect files without an active internet connection and I understood that this password based encryption ensures confidentiality by providing access only to authorised individuals.
