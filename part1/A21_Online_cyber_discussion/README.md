#A21 Participate in a an online cybersecurity discussion

As part of this activity, I participated in an online cyber security discussion in the online cyberguru server in discord. The discussion focused on phishing attacks, whether the attack is more directed to user responsibility or company responsibility, how these occur and ways to avoid or prevent them.

During the discussion I actively contributed by explaining key cyber security concepts like credential harvesting and social engineering. I highlighted that phishing attacks are often targeted against human behaviour more so than technical vulnerabilities which makes them particularly dangerous.I also emphasised the importance of multifactor authentication where there is a double layer of security and attackers would find a hard time hacking in even if our login credentials are leaked.Hence preventing unauthorised access.
