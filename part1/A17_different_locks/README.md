# A17 – Discover 10 different types of locks in use
## 1)Deadbolt lock
What I discovered – 
A Deadbolt lock is commonly used on homes and main entrance doors as it provides high security. It works on a solid metal bolt that that extends into the door frame when locked. Making forced entry very hard. It required a thumb turn or a key as in the evidence provided and does not rely on a spring mechanism. I discovered this at the main entrance of a medical clinic.
## 2)Keypad Combination lock
What I discovered – 
A keypad combination lock uses a numeric key as shown in the evidence below instead of a physical key and key hole. User enters the PIN on the keypad to unlock the door making it convenient and reducing risks of lost keys. This is very common in modern homes and offices and I discovered this in a company office.
## 3)keyed lever handle lock
What I discovered – 
This lock combines a door handle with a locking mechanism that requires a key. Its common at my house and is also often used in interior rooms or office spaces as well where access needs to be controlled.Its each to operate but provides less security that deadbolt lock.
## 4)electronic safe lock
What I discovered – 
An electronic safe lock is used to secure valuables inside safes. It requires a PIN code or digital input to unlock and may include a additional security features such as lockout after multiple failed attempts.It provides strong protection for sensitive assets such as cash or valuables.
## 5)Sliding patio door lock
What I discovered – 
This kind of lock is installed in sliding doors to protect them from being opened from outside. It secures the door by locking the sliding panel in place as discovered in the evidence. Some versions include additional bars or pins to enhance security.
## 6)Trilock
What I discovered – 
A Trilock is a combination of locking system commonly used in Australian homes. The lock was discovered at my home’s main entrance. It commonly integrates 3 locking mechanisms which include deadbolt lock, a latch and a snib lock.This provides multiple layers of security and convenience in a single system.
## 7)Combination padlock
What I discovered – 
 A combination padlock uses a rotating dial or number sequence instead of a key as show in the evidence below.It is commonly used for lockers, gates and storage units. It is convenient as there is not need to carry a key but the security depends on the strength of the code.
## 8)Cam lock
What I discovered – 
A cam lock is a simple lock used in cabinets, drawers and mailboxes.It operates using a key which turns the metal cam to secure or release the lock.It provides basic security for low risk items.
## 9)Thumb lock(privacy lock)
What I discovered – 
A thumb lock is mostly used in bathrooms and bedrooms for privacy. It can be locked from inside using a small turn mechanism. It is not designed for great security but prevents accidental or unwanted entry.
## 10)Smart Digital door lock
What I discovered –
A smart digital door lock allows users to unlock doors using a pin code, smartphone app or even biometric authentication such as finger print recognition. It eliminates the need for physical security keys and may include features like remote access, access logs and temporary access codes. This kind of lock is convenient and accessible to track authorised users.
