# A5 – Cryptographic Implementation used online

A very wide spread cryptographic implementation I discovered online is HTTPS. I learned that any secure website that used https protocol utilises TLS that is Transport Layer Security to secure connection between the user and the website. I accessed a secure website such as https://login.anz.com/internetbanking which is a banking site and uses https in its URL indicating the connection is encrypted. It can be identified that the website is secure through the sliders icon near the search bar. This makes sure that any data that is transmitted between the user and the website is not accessed by unauthorised entities. 

#### What was learned – 

HTTPS uses TLS which uses both Symmetric and asymmetric cryptography to secure communication. When the user connects to the website, the server presents a digital certificate to verify its identity. The browser then further establishes a secure connection using encryption keys, guaranteeing that all data transmitted is encrypted. TLS also ensures the data transmitted and received is exactly the same showcasing both integrity and confidentiality.
