# A11 – 5 Unique Access control devices
#### 1)Smart doorbell with mobile app access
I observed and used a smart doorbell system at my home where notifications are sent to my mobile phone when someone is at the door. And I can see the visitor and decide whether to allow access into the house or not. This demonstrates access control where access is granted through a mobile app.
#### 2) Garage remote
I use a garage remote to open and close my home garage to gain access to the house. This demonstrates access control as only authorised individuals with a garage remote can gain access to the house.
#### 3)Biometric authentication(fingerprint)
I use a Biometric Authentication system which requires my finger print to enter through the back door of the clinic. The system scans my fingerprint and compares it to stored finger prints to allow access to the building.
#### 4)Smart rider card scanner
I used my smart rider card at the train station to tag on before boarding. This is access control system as it verifies payment and lets access to the service.
