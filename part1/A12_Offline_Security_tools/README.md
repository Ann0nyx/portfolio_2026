# A12 – Discover 5 Unique Offline Security Tools

## 1. CCTV
I observed the CCTV cameras installed at my house which can be monitored through a screen inside my home. They help to continuously monitor activities surrounding our house and also helps in case of theft or security issues.

## 2. Biometric Authentication (Fingerprint)
I use my fingerprint to unlock my ipad and access by applications. It works offline and the system scans my fingerprint and compares it with stored fingerprint data to verify my identity.

## 3. Car Alarm System
I observed and experienced a car alarm system in my family vehicle. When the car is locked, it triggers an alarm if someone attempts to open the door without proper access or tampers with the vehicle.This acts as an intrusion detection system by alerting the user.

## 4. Locked Mailbox
I use a locked mailbox at home which requires a key to open it. This prevent others from unlocking our mail box or mails.


## 5. Laptop Password Lock
I use a password lock on my personal computer. It asks for a password whenever the laptop is turned on or wakes up from sleep making it secure for unauthorised entities.
